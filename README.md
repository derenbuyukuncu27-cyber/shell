# Mini Linux Shell

A simple Linux shell implementation written in C.

## Features

- Execute Linux commands
- Built-in cd command
- Built-in help
- Built-in clear
- Exit command
- Current working directory prompt
- Process management using fork() and execvp()

## Technologies

- C
- Linux
- POSIX System Calls

## Build

gcc main.c -o minishell

## Run

./minishell
